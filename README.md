# Teefax
Teefax ist ein Küchenwecker, der über die Konsole bedient wird. Als Parameter erwartet er die gewünschte Wartezeit in Sekunden. Nach Ablauf der Zeit ertönt ein Signal. 
Ich habe lange [Timerle](https://www.jfsoftware.de/timerle.htm) verwendet, um Tee zu kochen. Der Nachteil daran ist, dass man die Zeit jedes Mal neu einstellen muss. Mit Teefax kann man einfach eine Batch-Datei mit dem Befehl "teefax 180" oder eine Verknüpfung nutzen, um sofort 3 Minuten abzuzählen.

Neueste Version: [Herunterladen](https://github.com/Alsweider/Teefax/releases/latest)
